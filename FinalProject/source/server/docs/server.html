source.server

Module: source.server

Description: This module defines a TCPServer class for a simple multiplayer game server. The server accepts multiple client connections, assigns player IDs, and broadcasts game state information to connected clients.

  • TCPServer

    Declaration

    class TCPServer;

    Class: TCPServer

    The purpose of TCPServer is to accept multiple client connections. Each connected client has its own thread for server communication.

    • packetToBeSent

      Declaration

      char[Packet.sizeof] packetToBeSent(int[2][4] playerCoords, char[4] playerAssignment, int[2][2] ballCoords, char[80] message);

      Method: packetToBeSent

      Description: This method constructs a serialized packet to be sent to the server. It takes various game-related data, such as player coordinates, player assignments, ball coordinates, and a message, and constructs a packet to be sent to the server. The packet is then serialized using the serialize() method of the serverPacket struct.

      Parameters

      int[2][4] playerCoords

      A 2D array representing the coordinates of four players.

      char[4] playerAssignment

      An array representing player assignments (char[4]).

      int[2][2] ballCoords

      A 2D array representing the coordinates of two balls.

      char[80] message

      A string message with a maximum length of 80 characters.

      Return Value

      A serialized packet ready to be sent to the server.
